About This Digest

This is a personalized daily intelligence digest — automatically generated every morning from articles saved in Readwise Reader, analyzed by Claude (Anthropic's AI), and delivered by email and published here as a searchable archive.

How It Works

Each day, an automated script fetches articles from a Readwise Reader feed, filters them against a personal interest profile (companies, sectors, geographies, themes), and generates a structured analysis using Claude. The result is an email digest and an entry in this archive.

What It Covers

The digest is configured by the author's writing guide and filter config. These define which topics receive deep coverage, which entities trigger relevance alerts, and what analytical voice the output uses.
